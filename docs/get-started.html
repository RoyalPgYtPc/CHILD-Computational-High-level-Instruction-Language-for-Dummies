Follow these steps to install CHILD, set up the VS Code extension, and try interactive demos!

1. Download CHILD

Clone the repository or download the latest release:

git clone https://github.com/RoyalPgYtPc/CHILD-Computational-High-level-Instruction-Language-for-Dummies.git
2. Install Python

Make sure you have Python 3.6+ installed. Download from python.org.

python --version
3. Install the VS Code Extension
  1. Find CHILD-Computational-High-level-Instruction-Language-for-Dummies-1.0.0.vsix in the repo (main folder or releases).
  2. Open VS Code, press Ctrl+Shift+P and select Extensions: Install from VSIX...
  3. Choose the .vsix file you downloaded.

Note: The extension includes a built-in interpreter. You only need to install it once!

4. Run Your First Program

In VS Code: Open your hello.child file and click Run (‚ñ∂Ô∏è) at the top, or right-click and choose Run CHILD File.
Without VS Code? You can still use the standalone interpreter (child.py, also given in the repo):

python child.py hello.child
