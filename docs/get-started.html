Installation

Quick Setup Guide

Get CHILD running on your machine in just a few steps

1

Download CHILD

Clone the repository from GitHub to get the latest version of CHILD

git clone https://github.com/RoyalPgYtPc/CHILD-Computational-High-level-Instruction-Language-for-Dummies.git

Or download the ZIP file directly from the repository page

2

Install Python

CHILD requires Python 3.6 or higher. Download from python.org if you don't have it installed.

python --version
Note: Make sure Python is added to your system PATH during installation
3

Install VS Code Extension

Get professional IDE support with syntax highlighting and auto-completion

  1. Locate CHILD-Computational-High-level-Instruction-Language-for-Dummies-1.0.0.vsix in the repository
  2. Open VS Code and press Ctrl+Shift+P (or Cmd+Shift+P on Mac)
  3. Type "Extensions: Install from VSIX..." and select it
  4. Choose the .vsix file you downloaded
Pro Tip: The extension includes a built-in interpreter. You only need to install it once!
4

Run Your First Program

Create a file called hello.child and add your first CHILD code

python child.py hello.child

In VS Code: Simply click the Run button (▶️) or right-click and select "Run CHILD File"
