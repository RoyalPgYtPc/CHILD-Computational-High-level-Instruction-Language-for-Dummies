How It Works

From English to Machine Code

Watch how CHILD transforms your readable commands into executable code through a seamless pipeline

Step 01

You Write English

Type commands like "say Hello World" or "ask for name and remember it". No brackets, no semicolons, no confusion. Just natural language.

Step 02

CHILD Interpreter

Our intelligent parser reads your English commands and translates them into Python code instantly. All the heavy lifting happens behind the scenes.

Step 03

Python Execution

The generated Python code runs with full power and efficiency. Access to all Python libraries and ecosystem at your fingertips.

Step 04

Machine Execution

Your computer's processor executes the compiled machine code. Your ideas become reality, running at full native speed.

Comparison

See The Difference

Compare CHILD's intuitive syntax with traditional programming languages

CHILD - Natural Language
// Simple and readable say "Hello, World!" ask "What's your name?" and remember it as name say "Hello, " say name list [1, 2, 3, 4, 5] as numbers for each item in numbers as num say num end if name is "Alice" say "Welcome back!" end
Python - Traditional Syntax
# More complex syntax print("Hello, World!") name = input("What's your name? ") print(f"Hello, {name}") numbers = [1, 2, 3, 4, 5] for num in numbers: print(num) if name == "Alice": print("Welcome back!")
Get Started

Start Coding in 3 Steps

Get up and running with CHILD in less than 5 minutes

1

Clone the Repository

Download CHILD from GitHub to get started with the latest stable version

git clone https://github.com/RoyalPgYtPc/CHILD-Computational-High-level-Instruction-Language-for-Dummies.git
2

Install VS Code Extension

Get professional IDE support with syntax highlighting and intelligent auto-completion

code --install-extension child-language-1.0.0.vsix
3

Run Your First Program

Create a .child file and execute it with Python to see your code come to life

python child.py myprogram.child
