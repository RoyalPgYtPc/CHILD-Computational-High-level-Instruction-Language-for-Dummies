See It In Action

CHILD
// Simple and readable!
say "Hello, World!"

ask "What's your name?" and remember it as name
say "Hello, "
say name

list [1, 2, 3, 4, 5] as numbers

for each item in numbers as num
    say num
end
Python (Traditional)
# More complex syntax
print("Hello, World!")

name = input("What's your name? ")
print(f"Hello, {name}")

numbers = [1, 2, 3, 4, 5]

for num in numbers:
    print(num)

Get Started in 3 Steps

1 Download CHILD

Clone the repository or download the latest release:

git clone https://github.com/RoyalPgYtPc/CHILD-Computational-High-level-Instruction-Language-for-Dummies.git
2 Install the VS Code Extension
  1. Find CHILD-Computational-High-level-Instruction-Language-for-Dummies-1.0.0.vsix in the repo (main folder or releases).
  2. Open VS Code, press Ctrl+Shift+P and select Extensions: Install from VSIX...
  3. Choose the .vsix file you downloaded.

Note: The extension includes a built-in interpreter. You only need to install it once!

3 Write & Run Your First Program

In VS Code: Open your .child file and click Run (‚ñ∂Ô∏è) or right-click and choose Run CHILD File.
Without VS Code? Use the standalone interpreter (child.py, also in the repo):

python child.py myprogram.child
