See CHILD in Action

See It In Action

CHILD
// Simple and readable!
say "Hello, World!"

ask "What's your name?" and remember it as name
say "Hello, "
say name

list [1, 2, 3, 4, 5] as numbers

for each item in numbers as num
    say num
end
Python (Traditional)
# More complex syntax
print("Hello, World!")

name = input("What's your name? ")
print(f"Hello, {name}")

numbers = [1, 2, 3, 4, 5]

for num in numbers:
    print(num)

Get Started in 3 Steps

1 Download CHILD

Clone the repository or download the latest release

git clone https://github.com/RoyalPgYtPc/CHILD-Computational-High-level-Instruction-Language-for-Dummies.git

OR

1 Install VS Code Extension

Get syntax highlighting and auto-completion

code --install-extension child-language-1.0.0.vsix
2 Write Your First Program

Create a .child file and start coding!

python child.py myprogram.child
